# Varun Bhandari - Personal Portfolio

Professional portfolio website for Varun Bhandari, Senior Security Engineer and DevSecOps Expert.

## Features

- Responsive design that works on all devices
- Modern, clean UI with smooth animations
- Detailed professional experience timeline
- Comprehensive skills showcase
- Key achievements and projects
- Contact information and social links
- Optimized for GitHub Pages

## Tech Stack

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- Vanilla JavaScript
- Font Awesome Icons
- GitHub Pages Hosting

## Setup

1. Create a new repository named `varunbhandari.in` (or your preferred name)
2. Upload all files maintaining the folder structure
3. Add a CNAME file with your custom domain: `www.varunbhandari.in`
4. Enable GitHub Pages in repository settings
5. Configure DNS settings with your domain registrar:
   - Add A records pointing to GitHub Pages IPs:
     - 185.199.108.153
     - 185.199.109.153
     - 185.199.110.153
     - 185.199.111.153
   - Add CNAME record: www -> yourusername.github.io

## Customization

- Update content in `index.html`
- Modify colors and styles in `css/style.css`
- Adjust animations and interactions in `js/script.js`
